## 제한사항

출제진의 판단 하에 보내주신 문제가 다음 중 하나의 유형이라고 생각되는 경우 반려될 수 있습니다.

- 잘 알려진 문제
  - 단순한 메시지만 출력하면 되는 문제
  - 단순히 어떤 알고리즘을 구현하는 문제 -- 알고리즘이 잘 알려지지 않은 것일 경우는 제외
- 대회 목적에 부합하지 않는 문제
  - 넌센스 문제
  - 랜덤에 의존하는 문제
- 독창성이 부족한 문제
  - 다른 대회에서 등장한 적 있는 문제
